# Monetization Strategy

## Competitive Landscape

LoomFlow occupies a unique position in the creative tools market.

There is **no direct competitor** that combines:
- Organic, direction-field-following variable-width brush strokes
- Real-time animated preview (watching the image “paint itself”)
- Smart Layers with independent regional treatment
- Shared-palette batch processing
- Height-controlled brush width suitable for physical pen plotters

Closest existing apps fall into two categories:
- Simple artistic filters (Prisma, Waterlogue, Brushstroke, Oilist) — fast but low control, no real-time drawing animation, no variable brush height, no Smart Layers.
- Vector / tracing tools (Adobe Capture, Vectornator, Linearity Curve) — clean lines but lack organic brush feel and flowing color-direction awareness.

Desktop programs (Photoshop, Corel Painter, ArtRage) offer manual brushes but no automatic photo-to-stroke pipeline with the intelligent flow and coverage system we have.

This gap gives LoomFlow a strong differentiator: it feels like a **smart artistic collaborator** rather than just another filter.

**Pro** — **$9.99**
